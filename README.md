# Healthcare Coding Agent (Groq)

## Description
AI system that extracts diagnoses & procedures, maps ICD-10/CPT codes, and gives approval decision.

## Features
- LLM extraction (Groq)
- Rule-based validation
- Audit output

## Run
pip install -r requirements.txt
python app.py
